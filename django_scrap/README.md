> [!WARNING]
> 
> Przed uruchomieniem projektu, upewnij się, że utworzyłeś plik .env w głównym katalogu projektu i uzupełniłeś go odpowiednimi danymi. Poniżej znajdziesz przykładową zawartość pliku .env:
> 
## Przykładowa zawartość pliku .env

 Dane dostępowe do bazy danych PostgreSQL
> DB_USER=your_username
> 
> DB_PASSWORD=your_password
> 
> DB_HOST=your_host
> 
> DB_PORT=your_port
> 
> DB_NAME=your_database_name
